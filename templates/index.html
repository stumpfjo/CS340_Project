Home
Homepage with links to all subpages
Borrowers Home
Subpage to provide links to different Borrowers-related functions.
Add Borrowers
Page for manipulating the Borrowers table. Allows adding a new entry to Borrowers via a form.
View/Modify Borrowers
Allows displaying existing Borrowers in the system. The display of existing Borrowers can be searched/filtered by last_name, first_name, zip_code, and/or borrower_id. Displayed Borrowers can be updated or deleted.
View Borrower Checkouts
Displays the items checked out by a Borrower. Can be filtered to show only overdue items
Titles Home
Subpage to provide links to different Titles-related functions.
Add Titles
Page for adding new instances of Titles to the system
Search Titles
Allows searching the contents of the system by title.
Items Home
Subpage to provide links to different Items-related functions.
Checkouts
Page for processing checkouts of Items to Borrowers and returns of Items to the library.
Collection Management
Allows adding new Items to the library collection related to Titles in the system. Also allows removing Items from the library collection.
